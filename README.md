## My process

### Built with

- **HTML5 Semántico**: Estructura accesible y optimizada para SEO.
- **CSS Custom Properties**: Gestión eficiente de colores y fuentes mediante variables.
- **Flexbox & Grid**: Maquetación moderna para componentes y estructuras complejas.
- **Mobile-First Workflow**: Desarrollo priorizando dispositivos móviles (390px).
- **Metodología BEM**: Convención de nombres para un CSS escalable y organizado.
- **GitFlow & PRs**: Trabajo colaborativo real mediante ramas y revisión de código.
